# Market Researcher Memory

## Genealogy + Memorial Market (researched 2026-02-11)
- Global genealogy market: ~$5-7B (2025), CAGR ~11-12%, projected $16B by 2032
- Ancestry: $1B+ revenue, 3.6M paying subs, owns Find a Grave (acquired 2013)
- MyHeritage: ~$133M revenue (2018 figure), acquired by Francisco Partners 2021
- Familyspace.ru: Russian Ancestry equivalent, 700K-3M users, freemium, affordable
- VGD.ru: oldest Russian genealogy site
- Keeper Memorials: memorial + cemetery directions + family tree, freemium model
- EverLoved: YC-backed, <$5M revenue, funeral marketplace + free memorials
- NO competitor combines grave care services with genealogy features
- Russian diaspora in US: ~3M people
- Key user complaints: data accuracy, shared tree vandalism, no responses from contacts, subscription fatigue
